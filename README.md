# One 2nd

音声認識と筆談を組み合わせたコミュニケーションアプリ

## 概要

One 2ndは、言葉の壁を越えたコミュニケーションを実現する音声認識筆談アプリです。音声をテキストに変換し、11言語に対応した音声読み上げ機能を搭載しています。

## 主な機能

- 📝 リアルタイム音声認識
- 🗣️ 多言語音声読み上げ（11言語対応）
- 💬 対面でのスムーズなコミュニケーション
- 📱 シンプルで使いやすいUI
- 🌍 日本語、英語、韓国語、中国語、フランス語、スペイン語、ベンガル語、ヒンディー語、ロシア語、ポルトガル語、アラビア語に対応

## 対応プラットフォーム

- iOS 14.0以降
- macOS 11.0以降

## プライバシーポリシー

### データ収集について

One 2ndは、ユーザーのプライバシーを最優先に考えています。

**収集するデータ：**
- 本アプリは、個人を特定できる情報を収集しません
- 音声認識データは一時的に処理されますが、保存されません
- アプリ内の会話履歴はデバイス内にのみ保存され、外部には送信されません

**音声認識サービス：**
- 音声認識機能は、デバイス標準の音声認識APIおよびGoogle Speech-to-Text APIを使用します
- 音声データは認識処理のためのみ使用され、保存されません
- ベンガル語とヒンディー語の音声認識には、Google Cloud Speech-to-Text APIを使用します
- 音声データの取り扱いについては、各サービスのプライバシーポリシーに準拠します

**データの保存：**
- チャット履歴はお使いのデバイス内にのみ保存されます
- データは外部サーバーに送信されることはありません
- アプリを削除すると、すべてのデータが削除されます

**権限について：**
- マイク：音声認識機能に必要です
- カメラ（オプション）：プロフィール画像の設定に使用します
- インターネット接続：Google TTS/STT（一部言語）およびアプリの更新確認に使用します

**第三者サービス：**
- Google Cloud Speech-to-Text API（ベンガル語・ヒンディー語の音声認識）
- Google Cloud Text-to-Speech API（アラビア語・ベンガル語の音声合成）

これらのサービスの利用に関しては、Googleのプライバシーポリシーが適用されます：
[https://policies.google.com/privacy](https://policies.google.com/privacy)
